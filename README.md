# ai-image-rename
Rename all your images in a folder using Google Gemini.

Drop this rename_all_in_folder.sh script into a directory to rename all the images in it using Google Gemini.

1. Set your Gemini API Key.
2. Set the model you'll use by defining the correct GEMINI_API_URL.
3. Set your prompt to specify what you're looking for in each images.
4. Set the prepend and append text you want to add for each file.
5. Drop the script into the folder where your images live.
6. Run the script and your images will start getting renamed.

You can test this out by dropping the script into the demo-images folder in the repo.

Follow this guide to <a href="https://ai.google.dev/gemini-api/docs/api-key">get your Gemini API Key.</a>
